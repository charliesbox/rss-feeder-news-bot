# rss feeder news bot
my first telegram bot. uses feedparser to fetch latest news from different medias and hand them to the user with a link to a full article.  
a pretty useful news dispatcher to gather a variety of publications across the internet in one place. 

### features
- nyt (new york times) tech news fetching (i only added this one so far for a test)
- uhh..

### future features
- fetching news from different medias and their departments, such as tech or politics
- adding news to favorites (probably will require a dedicated database to store each user's favs)
- text formatting, with different emojis and overall smoother readability. some sort of design
- saving news every ~30 minutes to a database, as rss feeds always only contain the latest 20
